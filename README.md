# allegro2moe

Internal seller automation and listing optimization tool for Allegro marketplace.

## Purpose

This application is used for internal marketplace management tasks, including:

* seller offer management
* AI-assisted listing optimization
* title and description enhancement
* product image synchronization
* GPSR entity synchronization
* inventory and pricing automation
* marketplace content management

## API Usage

The application communicates with the official Allegro REST API using authorized seller credentials.

No public services or public API access are provided.

## Ownership

Private internal seller automation system.
